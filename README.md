Nfd
===
This is an alternative implementation of nfd, ndn forwarding daemon.

It is small (< 300 SLOC), and go get-able.

Install
=======
```
go get github.com/go-ndn/nfd
```

What is supported
=================
- [x] some control commands
- [x] forwarding
- [x] content store
- [ ] certificate
- [ ] routing
- [ ] ndn dns
